🚀 מדריך פרסום — NitPark

שלב אחר שלב לחנויות Google Play ו-App Store

1
התקן את הכלים הנדרשים
חד-פעמי · ~1 שעה
Node.js — הורד מ-nodejs.org אם עדיין לא מותקן
בדוק: node --version
Android Studio — הורד מ-developer.android.com/studio
גדול (~1GB) — תן לו להתקין את כל ה-SDK Components
ב-Android Studio: SDK Manager → Android 14 (API 34) — סמן ולחץ OK
הגדר משתנה סביבה: הוסף לנתיב המשתמש:
C:\Users\[שמך]\AppData\Local\Android\Sdk\platform-tools
2
הוסף פלטפורמת Android ל-Capacitor
בתיקיית NitPark בטרמינל
פתח טרמינל בתיקיית NitPark ורץ:
cd "C:\Users\ניתאי רביבו\Desktop\NitPark" npm install npx cap add android npx cap sync android
אם הכל הצליח, תופיע תיקיית android/ חדשה
פתח את הפרויקט ב-Android Studio:
npx cap open android
⚠️

ב-Android Studio — המתן ל-Gradle Sync לסיים (יכול לקחת 5-10 דקות בפעם הראשונה). לא לסגור!

3
הגדר App Icon וגרסה
ב-Android Studio
לחץ ימני על app/res → New → Image Asset
בחר Launcher Icons → העלה את icon-512.png שלך (512×512)
ב-android/app/build.gradle — עדכן גרסה:
versionCode 1 versionName "1.0.0"
4
בנה APK / AAB חתום
חשוב מאוד — שמור את ה-Keystore!
ב-Android Studio: Build → Generate Signed Bundle / APK
בחר Android App Bundle (.aab) — זה מה שGoogle Play דורש
לחץ Create new... ליצירת Keystore חדש:
• Key store path: שמור בספריית גיבוי מאובטחת
• Password: סיסמה חזקה — שמור אותה, לא ניתן לשחזר!
• Alias: nitpark-key
• Validity: 25 years
בחר Release → לחץ Create
🔑

קריטי: אם תאבד את ה-Keystore, לא תוכל לעדכן את האפליקציה בחנות לעולם! שמור אותו בענן (Google Drive / iCloud) + גיבוי מקומי.

5
העלה ל-Google Play Console
play.google.com/console
הירשם ב-Google Play Console ושלם $25 חד-פעמי
Create app → App name: "NitPark - חניה פרטית בקלות"
Default language: Hebrew (iw) · App type: App · Free or paid: Free
מלא את Store listing — כל הטקסטים כבר מוכנים ב-STORE-LISTING.md
העלה ל-Production → Create new release → Upload .aab
מלא את ה-Content rating questionnaire — בחר "Everyone"
לחץ Review release → Start rollout to Production
⏰

הביקורת לוקחת בדרך כלל 1-3 ימי עסקים לאפליקציה ראשונה. לאחר אישור — זמין בחנות תוך שעות.

AAB נבנה בהצלחה (Release)

חובה: Mac עם macOS 13+ ו-Xcode 15+. לא ניתן לבנות iOS app מ-Windows. אם אין לך Mac — ראה את הפתרון בסוף הסעיף.

1
הכן את ה-Mac
חד-פעמי
הורד Xcode מ-App Store (חינם, ~15GB)
הירשם ל-Apple Developer Program — $99/שנה
כתובת: developer.apple.com/programs
העתק את תיקיית NitPark למק (USB / Git / AirDrop)
הורד Node.js ל-Mac ורץ:
cd ~/Desktop/NitPark npm install npx cap add ios npx cap sync ios npx cap open ios
2
הגדר ב-Xcode
חשוב — Bundle ID + Signing
ב-Xcode — לחץ על App בצד שמאל → Signing & Capabilities
Team: בחר את חשבון ה-Apple Developer שלך
Bundle Identifier: il.co.nitpark.app
הוסף App Icon: גרור את icon-1024.png (1024×1024) ל-AppIcon ב-Assets.xcassets
Info.plist — הוסף הרשאות (Privacy strings):
NSLocationWhenInUseUsageDescription = "NitPark משתמשת במיקום כדי להציג חניות קרובות ולאפשר ניווט" NSCameraUsageDescription = "NitPark משתמשת במצלמה להעלאת תמונות החניה" NSPhotoLibraryUsageDescription = "NitPark ניגשת לגלריה להעלאת תמונות החניה"
3
Archive ו-Upload ל-App Store
ב-Xcode
שנה את Scheme ל-Any iOS Device (arm64) (לא Simulator)
Product → Archive — המתן לסיום (5-10 דקות)
חלון ה-Organizer יפתח → לחץ Distribute App → App Store Connect → Upload
כנס ל-appstoreconnect.apple.com → My Apps → NitPark
מלא פרטי App Store listing — הכל מוכן ב-STORE-LISTING.md
מלא את App Privacy (nutrition label): Location, Payment Info
לחץ Submit for Review
⏰

ביקורת Apple לוקחת בדרך כלל 24-48 שעות. לאפליקציות חדשות — לפעמים עד 7 ימים. שמור Demo Account מוכן (פרטים ב-STORE-LISTING.md).
